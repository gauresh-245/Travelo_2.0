# 🌍 Travelo 2.0

Travelo 2.0 is a full-stack travel planning and management platform designed to help users explore destinations, plan trips, manage itineraries, and access travel-related information through an interactive and user-friendly interface.

## 🚀 Features

* User-friendly travel planning interface
* Destination exploration and recommendations
* Trip itinerary management
* Transportation information management
* Responsive frontend design
* Backend API integration
* JSON-based travel data management
* Easy navigation and modern UI

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/gauresh-245/Travelo_2.0.git
```

### Navigate to Project Folder

```bash
cd Travelo_2.0
```

---

## Frontend Setup

```bash
cd Trip_Planner
npm install
npm start
```

Frontend will run at:

```text
http://localhost:3000
```

---

## Backend Setup

Open a new terminal:

```bash
cd backend
npm install
npm start
```

Backend server will start on the configured port.
